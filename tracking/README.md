# Tracking
- Express
- Passport
- Helmet
- Swagger
- Winston
- Morgan
- Prisma

## Init
```shell
yarn init -y
```

## Install
```shell
yarn add express
```

```shell
yarn add -D nodemon typescript @types/express @types/node ts-node
```

## Setup
```shell
tsc --init
```

## Configuration
Edit `"outDir": "./build"` and `"rootDir": "./src"` in `tsconfig.json`

## Prisma
### Init
```shell
npx prisma init
```

### Generate
```shell
npx prisma generate
```
